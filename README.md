# A nginx Debian / Ubuntu repository

## Installation

```
yarn install
```

## Usage

```
yarn dev
```

## Domain Change

Due to some reasons, we changed our domain and project name from `nginx.io` and `nginx.u.sb` to `n.wtf`.
